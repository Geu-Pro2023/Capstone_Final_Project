# Contributing to Titweng

Thank you for your interest in contributing to the Titweng Biometric Cattle Verification System!

## Development Setup

1. **Fork and Clone**
   ```bash
   git clone https://github.com/your-username/Capstone_Final_Project.git
   cd Capstone_Final_Project
   ```

2. **Backend Setup**
   ```bash
   cd backend
   python -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt
   ```

3. **Frontend Setup**
   ```bash
   cd admin_dashboard
   npm install
   npm run dev
   ```

4. **Mobile Setup**
   ```bash
   cd mobile_app
   flutter pub get
   flutter run
   ```

## Code Standards

- **Python**: Follow PEP 8
- **TypeScript**: Use ESLint configuration
- **Dart**: Follow Flutter style guide
- **Commits**: Use conventional commit format

## Pull Request Process

1. Create feature branch
2. Make changes with tests
3. Update documentation
4. Submit PR with description

## Issues and Bugs

Please use GitHub Issues with appropriate labels:
- `bug` for bugs
- `enhancement` for features
- `documentation` for docs
